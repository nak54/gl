# VS Code Shortcuts (Linux)

> Notes:
> - Linux keybindings are generally the same as Windows.
> - Some desktop environments/window managers may reserve shortcuts (e.g., `Alt+Left/Right`).

## General
- **Command Palette**: `Ctrl+Shift+P` or `F1`
- **Quick Open (files)**: `Ctrl+P`
- **New window**: `Ctrl+Shift+N`
- **Close window**: `Alt+F4`
- **Settings**: `Ctrl+,`
- **Keyboard shortcuts**: `Ctrl+K Ctrl+S`

## File / Editor
- **New file**: `Ctrl+N`
- **Open file**: `Ctrl+O`
- **Save**: `Ctrl+S`
- **Save as**: `Ctrl+Shift+S`
- **Save all**: `Ctrl+K S`
- **Close editor tab**: `Ctrl+W`
- **Reopen closed editor**: `Ctrl+Shift+T`
- **Next / previous editor tab**: `Ctrl+PageDown` / `Ctrl+PageUp`
- **Split editor**: `Ctrl+\`
- **Focus into 1st/2nd/3rd editor group**: `Ctrl+1` / `Ctrl+2` / `Ctrl+3`

## Navigation
- **Go to line**: `Ctrl+G`
- **Go to file**: `Ctrl+P`
- **Go to symbol in file**: `Ctrl+Shift+O`
- **Go to symbol in workspace**: `Ctrl+T`
- **Go to definition**: `F12`
- **Peek definition**: `Alt+F12`
- **Go back / forward**: `Alt+Left` / `Alt+Right`
- **Navigate editor group history**: `Ctrl+Alt+Left` / `Ctrl+Alt+Right`

## Search
- **Find**: `Ctrl+F`
- **Replace**: `Ctrl+H`
- **Find next / previous**: `F3` / `Shift+F3`
- **Find in files**: `Ctrl+Shift+F`
- **Replace in files**: `Ctrl+Shift+H`

## Editing
- **Cut line (empty selection)**: `Ctrl+X`
- **Copy line (empty selection)**: `Ctrl+C`
- **Move line up / down**: `Alt+Up` / `Alt+Down`
- **Copy line up / down**: `Shift+Alt+Up` / `Shift+Alt+Down`
- **Delete line**: `Ctrl+Shift+K`
- **Insert line below / above**: `Ctrl+Enter` / `Ctrl+Shift+Enter`
- **Undo / redo**: `Ctrl+Z` / `Ctrl+Y`
- **Toggle line comment**: `Ctrl+/`
- **Toggle block comment**: `Shift+Alt+A`
- **Format document**: `Shift+Alt+F`
- **Format selection**: `Ctrl+K Ctrl+F`
- **Indent / outdent line**: `Ctrl+]` / `Ctrl+[`
- **Fold / unfold region**: `Ctrl+Shift+[` / `Ctrl+Shift+]`
- **Fold all / unfold all**: `Ctrl+K Ctrl+0` / `Ctrl+K Ctrl+J`

## Multi-cursor / Selection
- **Add cursor above / below**: `Ctrl+Alt+Up` / `Ctrl+Alt+Down`
- **Add next match to selection**: `Ctrl+D`
- **Select all occurrences of current selection**: `Ctrl+Shift+L`
- **Expand / shrink selection**: `Shift+Alt+Right` / `Shift+Alt+Left`
- **Column (box) selection**: `Shift+Alt+Mouse Drag` (or `Shift+Alt+Arrow keys`)

## Panels / UI
- **Toggle Sidebar**: `Ctrl+B`
- **Toggle Panel (Terminal/Problems/Output/Debug Console)**: `Ctrl+J`
- **Explorer**: `Ctrl+Shift+E`
- **Search view**: `Ctrl+Shift+F`
- **Source Control view**: `Ctrl+Shift+G`
- **Run & Debug view**: `Ctrl+Shift+D`
- **Extensions view**: `Ctrl+Shift+X`

## Terminal
- **Toggle integrated terminal**: `` Ctrl+` ``
- **New terminal**: `Ctrl+Shift+`` ``
- **Split terminal**: `Ctrl+Shift+5`
- **Focus terminal**: `` Ctrl+` ``

## Debugging
- **Start/Continue**: `F5`
- **Stop**: `Shift+F5`
- **Step over**: `F10`
- **Step into**: `F11`
- **Step out**: `Shift+F11`
- **Toggle breakpoint**: `F9`

## Git (common)
- **Show Source Control**: `Ctrl+Shift+G`
- **View Git output**: (Command Palette) “Git: Show Git Output”
