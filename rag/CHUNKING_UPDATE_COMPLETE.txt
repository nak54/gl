================================================================================
KEY MODIFICATIONS
================================================================================

✅ 1. SEC MD&A Structure Recognition
   - Added recognition of 15 standard MD&A subsections
   - Subsections include: Overview, Results of Operations, Liquidity,
     Risk Factors, etc.
   - Uses intelligent pattern matching for fuzzy header detection
   - Falls back to size-based chunking if structure not found

✅ 2. 15% Overlap Implementation
   - Changed from fixed character overlap (e.g., 50) to percentage-based
   - Parameter: overlap=50 → overlap_percent=15
   - Calculated as: overlap_chars = (15/100) * (chunk_size * 4)
   - For 512-token chunks: ~307 characters overlap

✅ 3. Section-Aware Metadata
   - Each chunk now tracks its MD&A section
   - New field: TextChunk.section_name
   - Enables filtering chunks by topic/section
   - Preserved through embeddings and storage

✅ 4. Default Strategy Change
   - Default chunking strategy: 'size' → 'mda'
   - Uses SEC MD&A structure when available
   - Automatically falls back to size-based if needed
   - Backward compatible with existing code

✅ 5. New Methods Added
   - _find_mda_sections() - Detects MD&A boundaries
   - chunk_by_mda_structure() - Main MD&A chunking
   - _chunk_section_with_overlap() - Section-level chunking with overlap

================================================================================
USAGE EXAMPLES
================================================================================

Basic Usage (Recommended):

   from text_chunking import TextChunker

   chunker = TextChunker(
       chunk_size=512,          # Standard size
       overlap_percent=15       # 15% overlap (SEC-optimized)
   )

   chunks = chunker.run(documents)  # Uses 'mda' strategy by default

   # Access section metadata
   for chunk in chunks:
       print(f"Section: {chunk.section_name}")
       print(f"Text: {chunk.text[:100]}...")

Different Strategies:

   # MD&A structure (recommended)
   chunks = chunker.run(documents, strategy='mda')

   # Fixed-size chunking (fallback)
   chunks = chunker.run(documents, strategy='size')

Section Filtering:

   risk_chunks = [c for c in chunks if 'Risk' in c.section_name]
   financial_chunks = [c for c in chunks if 'Results' in c.section_name]
